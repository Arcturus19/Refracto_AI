# Refracto Vite + TypeScript

Scaffolded Vite React TypeScript project with Tailwind CSS and PWA support via `vite-plugin-pwa`.

Install dependencies (npm):

```bash
cd "C:/Users/VICTUS/Desktop/Refracto AI/refracto_vite"
npm install
```

Install required packages explicitly (if you prefer to install exact packages):

```bash
npm install react react-dom
npm install -D typescript @types/react @types/react-dom vite @vitejs/plugin-react tailwindcss postcss autoprefixer vite-plugin-pwa
```

Run dev server:

```bash
npm run dev
```

Build:

```bash
npm run build
```

Preview production build locally:

```bash
npm run preview
```

Notes:
- The PWA manifest uses theme color `#0ea5e9` (medical blue).
- Project files are under `src/` and scaffolding added for `components`, `pages`, `hooks`, `store`, `services`, and `layouts`.
